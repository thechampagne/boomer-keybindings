A patch that add camera movement keyboard shortcuts support for [Boomer](https://github.com/tsoding/boomer/) a zoomer application for Linux.


## Controls

| Control                                                                                         | Description                                                   |
|-------------------------------------------------------------------------------------------------|---------------------------------------------------------------|
| Drag with left mouse button or <kbd>h</kbd>/<kbd>j</kbd>/<kbd>k</kbd>/<kbd>l</kbd>              | Move the image around.                                        |
| Scroll wheel or <kbd>=</kbd>/<kbd>-</kbd> or <kbd>Ctrl</kbd> + <kbd>j</kbd>/<kbd>k</kbd>        | Zoom in/out.                                                  |
| <kbd>Ctrl</kbd> + Scroll wheel or <kbd>Ctrl</kbd> + <kbd>h</kbd>/<kbd>l</kbd>                   | Change the radious of the flaslight.                          |

## Configuration

New parameter:

| Name           | Description                                                                            |
|----------------|----------------------------------------------------------------------------------------|
| camera_speed   | How quickly camera move when using <kbd>h</kbd>/<kbd>j</kbd>/<kbd>k</kbd>/<kbd>l</kbd> |
